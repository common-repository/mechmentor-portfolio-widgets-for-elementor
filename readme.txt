MechMentor Portfolio is the Elementor Addon plugin used to build Portfolios and widgets

== Description ==

MechMentor Portfolio is the Elementor Addon used to create Portfolios.

Major features in MechMentor Portfolio include:

* Create Portfolio.
* Having various display options for portfolios.
* Mobile Optimized.

Features of **MechMentor Portfolio** plugin:

1. Add portfolio to any page of the website.
2. Easy to add images. 
3. Various displays option of the portfolio.
4. Can select various layouts to display portfolio. 
5. Optimize the images in portfolio on the go.
6. Create permalink for the portfolio. 
7. Can set the image height and width and column gaps also. 
8. Can set numbers of images per page.
9. Enable/Disable infinite scroll.


If you have any question or features request, please email us at mechdev72@gmail.com 
== Installation ==

#### From within WordPress

1. Visit 'Plugins > Add New'
2. Search for 'WP Pocket URLs'
3. Activate 'WP Pocket URLs' for WordPress from your 'Plugins' menu from WordPress.
4. Visit the new 'WP Pocket URLs' menu and click 'Add New' to add a new link.

#### Manually

1. Upload the mechmentor-portfolio-widgets-for-elementor/ folder to the /wp-content/plugins/ directory. 
2. Activate the plugin through the 'Plugins' menu in WordPress. 
3. Visit the new 'WP Pocket URLs' menu and click 'Add New' to add a new link.

== Frequently Asked Questions ==
= 1. How do I add the portfolio on the page? =
Go to elementor setting page and find the mechlin portfolio icon and click and drag to the page.

= 2. Can we set the different layout of the portfolio? =
Yes you can set the different layout of the portfolio.

= 3. Can I set custom image sizes of the portfolio? =
Yes you can set custom image sizes of the portfolio images.
